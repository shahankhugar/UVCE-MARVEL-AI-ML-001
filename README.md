## 📈 Task 5 — Linear Regression

Linear Regression finds the best-fitting line through data points to make predictions. Based on `y = mx + c`.

**What I built:** A model that predicts exam scores from study hours.

```
Dataset → Split (80/20) → Train → Evaluate → Predict
```

```python
model = LinearRegression()
model.fit(X_train, y_train)
model.predict([[7.5]])  # → ~78.5
```

| Metric | Purpose |
|--------|---------|
| R² Score | How well line fits (1.0 = perfect) |
| MSE | Average prediction error |

&gt; Linear Regression is the foundation of ML — the concept of minimizing error is behind ALL ML algorithms.

## 🧩 Task 6 — NumPy Matrix Puzzle

Decoded a scrambled matrix to reveal a hidden image using NumPy and Matplotlib.

## The Data
```python
data = np.load('encoded_array.npy')
# Shape: (200, 50) → Total: 10,000 elements
```

## Decoding the Clues

| Clue | Action | Code |
|------|--------|------|
| Reshape into a square | 10000 → 100×100 | `data.reshape(100, 100)` |
| Data might be sideways | Transpose | `reshaped.T` |
| End is the beginning | Flip | `np.flip(transposed)` |
```python
reshaped   = data.reshape(100, 100)
transposed = reshaped.T
flipped    = np.flip(transposed)

plt.imshow(flipped, cmap='gray')
plt.show()
```

## Result
The hidden image revealed was a **smiley face** 🙂
