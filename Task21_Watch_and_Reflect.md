## What is Machine Learning?

Machine Learning is teaching a computer to learn from data instead of manually writing rules for it.

In traditional programming, we write rules. In ML, the computer figures out the rules by looking at examples.

There are 3 types of ML:

- **Supervised Learning** — model learns from labelled data (most common)
- **Unsupervised Learning** — model finds hidden patterns on its own
- **Reinforcement Learning** — model learns by trial and error using rewards

---

## How is Data Prepared?

Raw data is always messy — missing values, duplicates, wrong formats. A model is only as good as the data it learns from.

The key steps are:

1. **Data Collection** — gather data from sources
2. **Data Cleaning** — fix missing values, remove duplicates
3. **Normalization** — scale all numbers to the same range
4. **Feature Selection** — keep only useful columns
5. **Train-Test Split** — 80% for training, 20% for testing

---

## My Takeaway

Data preparation is 80% of the work in any ML project. A good algorithm with bad data will fail. I also learned that ML is not magic — it's just pattern recognition at scale.
